# Position de l'ISS

Projet de Modélisation Numérique en Physique

*Damien Leprovost, Joséphine Maubian*

---

Objectifs actuels :
* Représenter sur un planisphère la trajectoire de l'ISS
* À partir de données collectées sur les jeux de données de la NASA
* Inférence d'orbite
